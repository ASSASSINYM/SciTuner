Osictuner is hight quality guitar tuner with real-time signal visualization.
Application use strong digital processing algorithm for frequency estimation and visualization plotting.
Oscituner presents many useful features:

• Simple and easy to use guitar tuner for electric or acoustic guitars
• Works in noisy conditions (background noise cancellation)
• Works with acoustic and electric real guitar
• Works for a 6-string real guitar
• Fully automatic mode (lets you tune quickly, hands-free string by string)
• Works also for some other instruments
• Actually listens and visualization plotting - works with the built-in microphone
• Ridiculously accurate digital signal processing
• Built by guitarist for guitarists (author use this application)
• Supports left-handed tuning (string has hint about thickness)
